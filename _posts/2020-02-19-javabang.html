1.프로젝트 개요

펜션 예약 대행, 정보 제공 서비스 플랫폼
숙소 이용자와 숙소 업체를 연결하는 플랫폼으로 승인된 업체로만 선별하여 업체의 과장광고를 최대한 해결하고 숙박 여정의 불편함을 해결하는 플랫폼 입니다.

2.조원구성 및 역할 분담

1) 공통 역할
기획, 정책 수립, db설계, 서류 작성, 웹 개발

2) 역할 분담
오건철(db관리자, 커뮤니티 게시판, 신고 기능, 게시글 관리(관리자), SMTP 메일 관리)
류별리(git 마스터, 일반검색, Map검색, FAQ/공지사항 게시판, 신고 기능, SMTP 메일 관리)
서현희(마이페이지, 결제 페이지)
김태엽(로그인/회원가입, 예매페이지)
최장원(업주/관리자 화면 구축, 리뷰 관리)
서선덕(펜션 관리(업주/관리자), 회원 관리)

3.개발 및 구현 환경

운운영체제 : Window OS
개발도구 : Eclipse
DBMS : Oracle DB – sqldeveloper
Server : Apache Tomcat 8.0
Language : Java, HTML5, CSS3, Javascript, jQuery, Servlet
디자인 툴 : Bootstrap
API : kakao 주소 검색 api, kakao map api
